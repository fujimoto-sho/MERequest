## 概要
- 動画編集依頼サービス
- どこよりもわかりやすく、使いやすいがコンセプト
- SPA（Vue.js、Vuex、Vue Routerを使用）
- APIはLaravelで作成

## 実装した機能
- ユーザ登録
- ログイン
- ログアウト
- 退会
- プロフィール編集
- 投稿作成
- 投稿修正
- 投稿削除
- 投稿一覧表示
- 投稿詳細表示
- 投稿検索
- パスワード変更
- パスワードリマインダー
- お気に入り登録
